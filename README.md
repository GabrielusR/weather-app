# Weather App
---

Simples aplicativo que se conecta a API em openweathermap.org e indica a temperatura, humidade relativa do ar, direção e velocidade do vento e tempo indicado por uma imagem. A consulta à essas informações é feita através de geolocalização buscando essas informações por latitude e longitude, caso o recurso de geolocalização não estiver disponível, uma caixa de diálogo é ativado perguntando pela cidade em que se consultar as informações metereológicas.

[Ver no browser](https://gabrielusr.github.io/weather-app/)
